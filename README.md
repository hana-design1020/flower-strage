# Salesforce Admin Practice Loop

TysonBlog の Salesforce 認定アドミニストレーター問題を、ローカルで繰り返し解くための練習ツールです。

## 使い方

```powershell
npm run scrape
```

その後、`index.html` をブラウザで開きます。

スマホからPCと同じWi-Fiなしで開く場合は、GitHub Pages などの無料HTTPSホスティングにこのフォルダの中身を置いてください。HTTPSで一度開くと、PWAのキャッシュによりオフラインでも使えるようになります。

## できること

- 問題のシャッフル
- 未回答、間違い、ブックマーク、苦手問題だけの出題
- 単一選択と複数選択に対応
- 回答履歴、正答率、連続正解数の保存
- 解説表示
- ローカルデータのリセット
- HTTPS配信時のオフライン利用

## データ更新

問題ページが更新された場合は、もう一度 `npm run scrape` を実行してください。生成先は `data/questions.js` です。

## 注意

このツールはローカル学習用です。取り込んだ問題データの扱いは、元サイトの利用条件に従ってください。
